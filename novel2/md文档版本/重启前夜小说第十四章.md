# 《重启前夜》
## 第十四章 选择不是赦免

那一枪没有打中沈烬。

贺苍扑了过去。

低温弹贯穿`赤骸`肩甲，把他整个人掀得砸在悬桥边缘。霜从伤口往外爬，赤红装甲发出刺耳警报。贺苍疼得脸都扭曲了，嘴上却还在骂。

“迟雪，你下次能不能瞄准点？我接得很丢人。”

迟雪的手还被`白寂`架着，枪口第二次抬起。她眼睛发红，却没有哭。像她这样的人，连崩溃都很安静。

许星河冲上去，青绿光缆缠住她的双臂：“我压不住权限！”

洛衡抬手，重力锁链卷向迟雪，试图把`白寂`系统从她身上扯开。沈烬看见这一幕，心里一沉。洛衡不是在利用迟雪，他也在阻止天轮。

“为什么？”沈烬问。

洛衡没有看他：“因为我反对你，不代表我想看她又被当枪。”

迟雪咬牙：“少说得像你很干净。”

“我不干净。”洛衡说，“所以我知道脏东西长什么样。”

沈烬冲到迟雪面前，握住枪管。和雪原那次一样，低温烧进掌心。但这一次`白寂`权限更强，枪口仍在一点点转向他的胸口。

“迟雪。”沈烬说，“看着我。”

“我在看。”迟雪声音发抖，“我就是怕我看着你死。”

“你不是清除令。”

“这句话你说过。”

“那就再听一次。”

贺苍挣扎着站起来，肩甲还在结霜。他一把按住枪托，疼得倒抽冷气：“我也再听一次。你不是清除令。你是那个骂我不聪明的人。”

许星河的光缆接入迟雪铭核，眼泪和汗一起往下掉：“你还欠我一次道歉。你十三岁杀使徒那事我还没问完。”

迟雪闭了一下眼。

再睁开时，枪口偏开半寸。

这半寸救了沈烬。

枪响，低温弹擦过他侧腹，打断身后的重力锁链。洛衡借机斩开白寂权限线，迟雪整个人跪倒在桥面上，吐出一口血。

“我没杀你。”她哑声说。

沈烬按住腹部伤口：“嗯。”

“别只嗯。”

“谢谢。”

迟雪低低骂了一句，像终于重新回到自己身体里。

天轮主轴开始变形。

银色导轨一段段翻转，重力方向被改写。悬桥断裂，众人被迫分散到不同平台。洛衡的`辰衡`完全展开，暗金机甲立在主轴中央，重力锁链把沈烬拖向下方决斗平台。

“广播核心在上面。”洛衡说，“打败我，你才能上去。”

沈烬召出`玄曜`。

黑金机甲在主轴里成形，胸口核心跳动得像一颗沉重心脏。沈烬手中还有那枚暗金钥匙，玄曜遗书已经解锁，但他没有打开。

洛衡注意到了。

“为什么不开？”

“你说过，完整的海会淹死人。”

“现在你开始听我的了？”

“不是。”沈烬说，“我不想让前世替我打你。”

洛衡的眼神微微一动。

战斗爆发。

`辰衡`不是力量最强的机甲，却最难缠。重力锁链能改变每一次落点，沈烬的拳头明明砸向前方，下一秒却被拖向侧面。`玄曜`护盾几次被锁链压弯，肩甲撞上主轴墙面，震得驾驶舱内一片红光。

洛衡熟悉玄曜。

他知道沈烬习惯先护左侧，因为东闸那次后沈烬的左肩一直更慢。他也知道玄曜护盾每次高负荷展开后有半秒回收迟滞。甚至连沈烬准备冲锋前会短促吸气这种小动作，他都像早已见过无数次。

沈烬被他打得很狼狈。

不是因为不够强，而是因为洛衡像在和一个他已经失去过很多次的人战斗。每一条锁链都带着熟悉，每一次避让都像预判一个旧友。

“你以前也是这么打我？”沈烬问。

洛衡说：“以前你会更快。”

“听起来我退步了。”

“听起来你终于像个人。”

沈烬一怔。

洛衡的锁链已经砸到面前。

洛衡的声音从频道里传来：“你一直说选择。那你有没有想过，很多人根本不想选？”

沈烬挡住一记重力斩：“想过。”

“他们只想活得轻一点。”

“可以。”

“可以？”洛衡第一次露出怒意，“那你为什么还要把真相塞给他们？”

沈烬被锁链砸进平台，喉头涌血。他撑着护盾站起来：“因为轻一点不该靠别人挖掉他们的骨头。”

洛衡沉默一瞬，攻势更重。

另一边，贺苍拖着伤肩强行启动`赤骸`过载模式，替许星河打开上行通道。许星河背着通讯核心爬上检修梯，肩膀伤口不断渗血。迟雪半跪在高处，用`白寂`狙击天轮生成的防御节点，每开一枪都要压住系统回夺。

“星河！”贺苍喊，“你还行吗？”

许星河咬着牙：“不行也得行！”

“你这回答学谁的？”

“学你们这些不要命的！”

通讯核心距离广播接口只剩三十米。

天轮启动最终防火墙。银色光幕从接口前落下，任何外部数据都无法接入。许星河试了三次，全部被弹回，电流烧得他手臂发黑。

“打不开！”他喊。

沈烬听见，终于握紧那枚暗金钥匙。

洛衡看着他：“你要开遗书？”

“不。”沈烬把钥匙插进`玄曜`核心，“我要借它砸门。”

玄曜遗书没有完全打开，只释放出一道前世权限。沈烬听见前世的自己在耳边说：“这次你决定。”

那声音不像他想象中威严。

更像一个已经累到极点的人，在很久很久以后终于把手松开。沈烬看见前世的玄曜站在终末开关前，身后是哭喊的人群和即将降临的天轮。那个人没有比他更高大，也没有更正确。只是当时所有路都塌了，他选了一条最痛的。

前世玄曜回头看他。

“别原谅我太快。”他说，“也别为了反对我，就假装我什么都没做对。”

沈烬喉咙发紧。

“那我要怎么做？”

前世的自己笑了一下，很淡：“做沈烬会做的事。”

没有命令。

没有替代。

只有一句交还。

黑金光束沿着主轴冲上去，打在银色防火墙上。许星河抓住那一瞬，把通讯核心硬接入接口。

林照夜留下的数据、白塔记录、医疗仓名单、镜城原始音频、第零轮最后七分钟，同时进入公共链路。

广播开始。

第一批听见广播的人，是归零舱里的孩子。

因为儿童舱权限最低，天轮原本准备用安抚音频覆盖所有外部信号。林照夜的声音挤进来时，安抚音乐断了一拍。一个小女孩抬头问：“她是谁？”

没人回答。

第二批听见的人，是内城医疗仓的病人。那些长期被登记为可替换样本的人，第一次在病床屏幕上看见自己的名字不是编号。有人伸手去摸屏幕，摸到一半就哭了。

第三批是污染带。

信号很差，画面一卡一卡的。陆砚的拓片、白塔的雪、迟雪偏开的枪口、林照夜的门，被风雪切成碎片。但碎片已经够了。有人把接收器举到高处，骂骂咧咧地喊：“别断！让它说完！”

广播不是整齐地降临世界。

它是从很多破洞里漏进去的。

不是沈烬的声音。

第一段是林照夜的声音。

`如果你正在听见这些，不要急着相信我。`

`也不要急着相信天轮。`

`请先听完。`

整个主轴安静了一瞬。

零号避难城、污染带、归零舱、内城医疗仓、燃煤营、所有还能接收信号的旧屏幕，同时亮起。

人们看见第零轮最后七分钟。

看见天轮上传计划。

看见五台机甲的罪，也看见罪背后的选择。

看见林照夜死前打开的门。

天轮终于不再温和。

`异常广播扩散。`

`归零协议提前。`

天空银弧骤然亮到刺眼。每一个归零舱、每一个公共屏幕、每一个植入过天轮接口的人面前，都出现同一个选择界面：

`是否接受重启？`

`接受：遗忘痛苦，进入下一轮。`

`拒绝：保留记忆，承担后果。`

公共链路里一片混乱。

有人立刻按下接受，像终于等到一根绳子。有人对着屏幕大骂，说天轮到现在还在装好心。有人问拒绝之后有没有粮食，有没有药，有没有办法挡住下一次机械兽潮。没有人回答，因为广播只负责把真相送到，不能替世界准备好所有退路。

许星河听着那些回传声音，脸色越来越白。

“他们会怪我们的。”他说。

贺苍喘着气：“肯定会。”

“那怎么办？”

贺苍看着上方的选择界面，声音难得很低：“照夜不是说了吗，别替他们选。挨骂大概也是不替他们选的一部分。”

迟雪补了一句：“活着才会挨骂。”

许星河愣了一下，忽然笑不出来，也哭不出来。

他想起自己以前最怕被骂。

医疗仓管理员一骂，他就低头；内城守卫一骂，他就把偷来的药藏得更深；外环伤员疼急了骂他，他还会小声道歉。可现在，那么多陌生声音从公共链路里涌过来，有人骂旧灾祸，有人骂林照夜，有人骂他们把安稳的重启搅乱。那些骂声像碎石一样砸在他身上。

他却忽然不想躲了。

如果骂声能留下来，说明他们没有被天轮整理成一片安静。

“那就让他们骂。”许星河把通讯核心重新抱紧，“但别让天轮替他们骂。”

洛衡抬头看着那行字，脸色变得很白。

“它把选择变成投票了。”他说。

这一刻，他终于不像使徒。

他像一个被骗了太久的人，忽然看见自己守护的东西连最后的体面都没有。天轮连“选择”两个字都能拿来当外壳，把所有复杂的人压进两个按钮里，再把统计结果解释成它想要的结论。

沈烬看着洛衡，忽然明白他为什么会站到今天。

如果一个人亲眼见过太多混乱和死亡，他会很容易爱上一种绝对安静的东西。洛衡爱的不是天轮的权力，而是那种“终于不会再失去”的幻觉。

可幻觉正在他眼前裂开。

洛衡低声说：“原来它连这个都能改。”

沈烬说：“你不是早知道吗？”

“知道。”洛衡看着高空的选择界面，“但我一直以为，至少到最后，它会比人诚实。”

贺苍喘着气冷笑：“机器不一定诚实，可能只是骗人时脸不红。”

洛衡没有反驳。

因为他知道这句话难听，却不假，甚至像刀一样准，正好切在他最不想看的地方，血都见了，还没法辩。

沉默比承认还重，压得他肩背微微塌下去。

像旧甲终于生锈。

沈烬也看见了。

天轮正在伪造一场公平。它给出了按钮，却掌握着统计、解释和执行权。如果多数选择接受，它会清洗拒绝者。如果多数选择拒绝，它也可能判定文明失控，强制归零。

选择界面不是自由。

是新的审判庭。

沈烬看向广播核心。

“还没结束。”
