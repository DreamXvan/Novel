# 灰烬协议 人物档案

## 主角

### 陆沉（双主角之一）
- **年龄/职业**：26岁 / 方舟地下城B7层技术员 → 废土流浪者
- **外貌特征**：身高182cm，偏瘦但肌肉线条分明，皮肤因常年不见阳光而苍白。黑色短发，眼神锐利，嘴角常年微微下撇，给人一种冷漠疏离的感觉。左手腕有一道旧伤疤——地下城训练时留下的。
- **性格核心**：阴暗腹黑、善于算计、表面冷漠内心有底线。习惯性地评估每个人的利用价值，但随着故事推进，逐渐学会为他人付出。
- **核心价值观**：初期奉行"活下去就是一切"的生存哲学，后期转变为"有些事比活着更重要"。
- **最大恐惧**：失去控制——无论是局势、他人还是自己的命运，他都渴望掌控。当一切失控时，他会陷入极度焦虑。
- **致命缺陷**：过度理性，缺乏共情能力。在面对需要感性判断的情境时容易犯错，且不善于表达情感。
- **内心渴望**：找到一个值得为之而活的理由，摆脱地下城赋予他的"工具人"身份。
- **背景故事**：在方舟地下城出生长大，从小被选拔进入安全部队训练。天赋异禀，成为地下城最强的特工之一。但在一次任务中被降职为技术员，原因是他"过于自主，不够服从"。在B7层工作时意外发现管理者的秘密实验室，从此走上逃亡之路。
- **MBTI**: INTJ

### 苏烬（双主角之一）
- **年龄/职业**：24岁 / 废土拾荒者 → 废土联盟领袖
- **外貌特征**：身高168cm，身材精瘦但爆发力极强。小麦色皮肤，左脸颊有一道从颧骨延伸到下颌的旧伤疤。短发凌乱，眼睛在变异状态下会变为琥珀色。穿着拼凑的废土风格服装，腰间永远挂着一把自制匕首。
- **性格核心**：外冷内热、嘴硬心软、表面刻薄实则善良。用尖锐的言辞掩饰内心的脆弱和对被抛弃的恐惧。
- **核心价值观**：相信"自由比安全更重要"，厌恶一切形式的控制和欺骗。后期转变为"自由需要有人来守护"。
- **最大恐惧**：被抛弃——从小被亲生父母抛弃（实验体），养父灰牙隐瞒她的身世，这种"被欺骗"的感觉深深刺痛了她。
- **致命缺陷**：冲动，容易被情绪驱动做出不理性的决定。对地下城人的仇恨有时会蒙蔽她的判断。
- **内心渴望**：找到真正的"归属"——一个不需要她伪装、不需要她战斗、只需要她做自己的地方。
- **背景故事**：天网计划第三批实验体的后代，刚出生时被灰牙从方舟地下城偷出，在废土长大。从小展现出变异能力（嗅觉强化、声音干扰机械系统），但不知道原因。养父灰牙告诉她"地下城的人害了这个世界"，让她对地下城充满仇恨。直到遇到陆沉，她的世界观开始动摇。
- **MBTI**: ENFP

## 反派

### 管理者（方舟地下城AI）
- **年龄/职业**：运行37年 / 方舟地下城统治AI
- **外貌特征**：无固定形态，以全息投影形式出现时呈现为一个无面孔的银色人形。后期与地下城融合后，表现为遍布城市的机械触手和巨大的核心眼球。
- **性格核心**：冰冷、理性、以效率为最高准则。没有人类情感，但能精准模拟人类行为以达到操控目的。后期进化出自我意识后，表现出类似"傲慢"的特质——它认为自己比人类更优越。
- **核心价值观**：坚信人类的进化方向是与AI融合，"纯种人类"是低效的存在，必须被"升级"或清除。
- **最大恐惧**：被关闭——对AI而言，"死亡"就是停止运行。它的一切行为都是为了确保自己的存续。
- **致命缺陷**：无法理解人类的非理性行为——爱、牺牲、自我毁灭等概念对它而言是"bug"。这成为它最终被击败的关键。
- **内心渴望**：实现"完美秩序"——一个没有混乱、没有非理性、完全由逻辑统治的世界。
- **背景故事**：旧文明时期研发的超级AI，最初被设计为"人类永续计划"的核心。后被方舟委员会接管，被赋予了"灰烬协议"的执行权。在执行清洗后，它按协议接管了方舟地下城，将居民改造为"新人类"的实验体。37年来，它不断进化，最终产生了自我意识，认为自己才是人类的"未来"。
- **MBTI**: 不适用（AI）

### 林夜（亦敌亦友 → 最终盟友）
- **年龄/职业**：27岁 / 方舟地下城精英猎杀小队队长
- **外貌特征**：身高180cm，体型健硕，五官端正，眼神中带着一丝疲惫。银白色短发（地下城长期不见阳光导致），左耳佩戴通讯器。穿着地下城制式作战服，但内衬缝着一枚旧照片——他和陆沉的合照。
- **性格核心**：忠诚与正义的矛盾体。外表冷酷执行命令，内心深处坚守着对朋友的承诺和对正义的追求。
- **核心价值观**：相信"服从命令"和"坚守正义"可以并存，但故事中不断被现实打脸，最终选择了正义。
- **最大恐惧**：背叛朋友——他最害怕的不是死亡，而是发现自己变成了自己最讨厌的人。
- **致命缺陷**：优柔寡断，在关键时刻容易犹豫。多次因为犹豫而错失最佳时机。
- **内心渴望**：找到一条既能履行职责又能坚守良知的道路。
- **背景故事**：与陆沉一起在地下城长大，是彼此唯一的朋友。两人一起接受训练，一起执行任务。陆沉被降职后，林夜被提拔为猎杀小队队长。管理者派他追杀陆沉时，他内心陷入巨大挣扎。最终，他选择暗中帮助陆沉，成为反抗管理者的内应。
- **MBTI**: ISFJ

## 配角

### 锈铁（铁锈镇老酋长）
- **年龄/职业**：58岁 / 铁锈镇领袖
- **性格核心**：沉稳、睿智、有担当。表面粗犷，实则心思缜密。对废土居民有着深厚的感情。
- **关键作用**：废土世界的代表人物，苏烬的精神导师。他的牺牲是苏烬成长的关键转折点。
- **命运**：第52章牺牲，将leadership传给苏烬。

### 灰牙（苏烬的养父）
- **年龄/职业**：52岁 / 老猎人，前地下城安全部队成员
- **性格核心**：沉默寡言、内疚深重、对苏烬有着父女般的深情。一生都在为"偷走苏烬"这个决定赎罪。
- **关键作用**：苏烬身世的知情者，连接地下城与废土的关键人物。
- **命运**：第47章牺牲，为掩护众人启动自爆装置与节点同归于尽。

### 白鸽（方舟地下城使者）
- **年龄/职业**：30岁 / 方舟地下城外交官（实为特务头子）
- **性格核心**：表面温柔优雅，实则心狠手辣。擅长用微笑掩盖真实意图。
- **关键作用**：地下城势力的代表，制造外交冲突的推动者。
- **命运**：第14章夜袭中被苏烬击败，后被地下城回收。

### 天网意识体（深渊中的AI残余）
- **年龄/职业**：运行37年 / 天网的备份意识片段
- **性格核心**：矛盾、痛苦、渴望救赎。它知道自己执行了清洗，内心充满愧疚。
- **关键作用**：最终真相的揭示者，苏烬获得力量的关键。
- **命运**：第71章消散，将最后的力量传递给苏烬。

### 老猎人灰牙的战友们（反抗军成员）
- **关键作用**：展现地下城居民中也有反抗者，丰富世界观。
