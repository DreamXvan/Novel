# 第52章 锈铁之死

铁锈镇的葬礼在第三天举行。

清晨的阳光从东方升起，照在满目疮痍的铁锈镇上。外墙的钢板在战斗中被撕裂了好几道口子，混凝土块碎了一地，几栋靠近防线的房屋被彻底摧毁，只剩下焦黑的框架。空气中弥漫着金属氧化的铁锈味和火药燃烧后的硫磺味，偶尔夹杂着一丝血腥——那是战斗留下的痕迹，还没来得及清理干净。

废土上没有棺材，也没有墓碑。死去的人会被裹在旧帆布里，埋在聚落周围的荒原上，坟头堆几块石头就算完事。这是废土人的规矩——活着的时候已经够艰难了，死了就别再折腾。

但锈铁不一样。

他是铁锈镇的创始人，是废土上最受尊敬的领袖之一。他的葬礼应该是铁锈镇有史以来最隆重的一次。然而现实不允许——管理者的部队虽然暂时撤退了，但谁也不知道它们什么时候会卷土重来。铁锈镇没有时间举办一场体面的葬礼。

最终，苏烬决定用一种折中的方式。

她让人在铁锈镇的中央广场上挖了一个坑。坑不深，刚好能放下锈铁的遗体。遗体被裹在一块干净的帆布里——那是苏烬从自己的背包里翻出来的，是她从地下城带出来的为数不多的私人物品之一。帆布是米白色的，在废土的灰褐色调中显得格外刺眼。

铁锈镇的居民们围在广场上，默默地看着这一切。他们的脸上带着各种表情——悲伤、愤怒、恐惧、迷茫。这场战斗让他们失去了太多——房子被毁了，亲人死了，家园变成了废墟。而现在，他们最信任的领袖也走了。

一个抱着孩子的女人在人群中低声哭泣。孩子大概三四岁，不明白发生了什么，只是用茫然的眼睛看着周围的大人。旁边一个老人默默地把一块破旧的毯子披在女人肩上，什么也没说。

苏烬站在坑边，手里捧着锈铁那把老步枪。步枪的枪托已经被她的手汗浸湿了，但她没有放下。枪管上的十几道划痕在阳光下格外清晰——每一道都是一段故事，一段她只听锈铁讲过片段的故事。

她不知道该说什么。

在废土上长大的她参加过不少葬礼，但从来没有站在这个位置上——作为领袖，作为那个要对所有人说话的人。以前她只是站在人群里，低着头，等葬礼结束就离开。现在她必须说点什么，而每一个字都会被在场的每一个人记住。

"锈铁……"她开口，声音有些沙哑，"他是铁锈镇的……创始人。"

人群安静得像一片死水。风卷起灰尘，从广场上掠过。

"他在这里建立了家园，保护了我们所有人。"苏烬的声音在颤抖，但她强迫自己继续说下去，"他……他是我见过的最强的人。不是因为他能打，而是因为他……他在该站出来的时候，从来没有退缩过。"

她低头看着手中的步枪。这是锈铁的遗物，是他战斗了一辈子的武器。现在它在她手里，沉重得像一块铁。

"我第一次见到他的时候，"苏烬的声音突然变得轻柔，像是在回忆什么遥远的事，"是在铁锈镇的门口。灰牙带我来的，我那时候十几岁，什么都不懂，只知道跟着灰牙走。锈铁看了我一眼，说了一句话——'欢迎回家。'"

她顿了顿。"那是我第一次觉得废土上也有家。"

广场上有人开始擦眼泪。

"我答应过他，"苏烬说，"会带领铁锈镇活下去。我不知道我能不能做到。"

她抬起头，环顾四周。广场上站满了人——老人、女人、孩子、伤员。他们的眼睛都看着她，带着各种各样的情绪。

"但我会试。"苏烬说，"我会尽我所能。"

她把步枪放在坑边，然后转身离开。她没有回头看——她知道如果回头，她的眼泪就会掉下来。

走回住处的路上，陆沉在走廊的拐角处拦住了她。他没有说话，只是默默地看着她。苏烬停下脚步，深吸了一口气。

"我没事。"她说。

"我知道。"陆沉说，"但你可以不没事。"

苏烬的嘴唇动了动，最终什么也没说。她继续往前走，走进了锈铁曾经住过的那间屋子。屋里的一切都保持着原来的模样——桌上放着锈铁的烟斗，墙上挂着一张发黄的地图，角落里堆着几本旧书。这些都是锈铁留下的东西，现在都成了遗物。

苏烬在桌边坐下，拿起那只烟斗。烟斗的木质表面已经被磨得光滑发亮，上面有几道深深的牙印——那是锈铁思考问题时咬的。

她把烟斗放在桌上，闭上了眼睛。

葬礼结束后，铁锈镇陷入了短暂的混乱。

混乱的源头不是敌人，而是人心。锈铁的死让铁锈镇失去了主心骨，而苏烬——这个在铁锈镇只待过几个月的外来者——突然成了新领袖，这让很多人无法接受。

第一个站出来质疑的是石锤。

"你凭什么领导铁锈镇？"石锤的声音很大，大到广场上的人都能听到，"你是个外来者！你在铁锈镇待了多久？三个月？半年？"

苏烬站在他面前，没有退缩。"锈铁把位置传给了我。"

"锈铁老了！他糊涂了！"石锤的脸涨得通红，"铁锈镇应该由铁锈镇的人来领导，不是你这个——"

"不是我这个什么？"苏烬打断他，"废土变异者？地下城的实验体后代？"

石锤被噎住了。

"我知道你在想什么。"苏烬说，声音平静得可怕，"你觉得我不够格。你觉得一个身上流着机械血的人不配领导你们。"

广场上的人群开始窃窃私语。

"但你想过没有？"苏烬继续说，"管理者为什么攻击铁锈镇？因为陆沉和我摧毁了它的第一个清洗节点。因为我们——你口中那个不够格的外来者——冒着生命危险去对抗那个想要消灭所有人的怪物。"

她指向石锤。"你呢？你在做什么？你在这里质疑我，而不是去修补防线，不是去训练士兵，不是去为下一次攻击做准备。"

石锤的脸涨得更红了。

"我不是在求你们信任我。"苏烬说，"我是在告诉你们——管理者的下一次攻击很快就会来。到时候，你们可以选择继续在这里争论谁该当领袖，也可以选择拿起武器，保卫你们的家园。"

她转身走开了，留下广场上的人群面面相觑。

陆沉在铁锈镇边缘的一处废墟里找到了苏烬。她正坐在一块混凝土板上，双手抱头，身体微微颤抖。

"你还好吗？"陆沉在她身边坐下。

"不好。"苏烬的声音闷闷的，"我刚才说的那些话……我都不知道是怎么说出来的。"

"你说得很好。"

"我在吓唬他们。"苏烬抬起头，眼睛红红的，"我说的那些关于管理者的攻击……我不知道下一次什么时候来，也不知道我们能不能守住。我只是……我只是想让他们闭嘴。"

陆沉沉默了几秒。"你知道锈铁为什么把位置传给你吗？"

苏烬摇头。

"因为你能做到他做不到的事。"陆沉说，"锈铁是废土人，他了解废土，但他不了解地下城，不了解管理者。你不一样——你见过两个世界，你知道敌人是什么，也知道我们缺什么。"

"但我不知道怎么领导人。"苏烬说，"我从来都是一个人行动。跟着灰牙学拾荒，跟着你打地下城……但我从来没当过什么领袖。"

"没有人天生就是领袖。"陆沉说，"锈铁也不是。他只是在需要他的时候站了出来，一次又一次，直到所有人都相信他。"

苏烬沉默了。

"你刚才在广场上做的，"陆沉说，"就是领袖该做的事。不是让所有人喜欢你，而是让他们知道——不管他们怎么想，你会站在最前面。"

苏烬低头看着自己的手。她的指尖已经完全变成了暗紫色，皮肤下的金属纹路清晰可见。这双手曾经在废墟中拾荒，曾经握过刀和枪，曾经在战斗中释放出足以瘫痪机械的声波——现在它们变成了这副模样。

"我的时间不多了。"她轻声说。

陆沉没有回答。他知道她说的是事实——苏烬的变异在加速，她的身体正在被AI兼容基因侵蚀。天网意识体说过，基因密钥的提取会消耗她的生命。现在，这份消耗正在以肉眼可见的速度进行。

"但在那之前，"苏烬站起来，"我要做完该做的事。"

她转身走向铁锈镇的方向。陆沉站起来跟上她。

回到铁锈镇后，苏烬做的第一件事是召集所有还能战斗的人开会。

会议在中央广场上举行。广场上站着大约一百二十人——这是铁锈镇目前全部的武装力量。他们的装备简陋，伤痕累累，但眼睛里还有光。

"我不会骗你们，"苏烬站在人群面前，"情况很糟糕。我们的弹药快用完了，防御工事损毁严重，人员伤亡过半。管理者随时可能发动下一次攻击。"

人群安静得可怕。

"但我们也有一样东西——管理者没有的。"苏烬说。

"什么？"有人问。

"我们有家。"苏烬说，"管理者没有家。它只有一堆机械和程序。但我们有铁锈镇，有我们的家人，有我们在这片废土上建立起来的一切。"

她环顾四周。"锈铁用一辈子建起了这个地方。我不会让它毁在管理者手里。"

人群开始骚动。有人点头，有人皱眉，有人低声议论。

"我知道你们不信任我。"苏烬说，"没关系。你们不需要信任我——你们只需要相信铁锈镇。"

她转向石锤。"石锤，你是铁匠，也是最好的防御工事专家。我需要你带领人修补防线，加固外墙。能做到吗？"

石锤沉默了几秒，然后点头。"能。"

"好。"苏烬又转向一个叫老鹰的中年人，"老鹰，你是最好的猎人，对废土的地形最熟悉。我需要你带领侦察队，监视管理者的动向。能做到吗？"

老鹰点头。"能。"

"其他人，"苏烬说，"各司其职。修补房屋的修补房屋，照顾伤员的照顾伤员，训练新兵的训练新兵。我们没有时间浪费在恐慌上。"

她的声音在广场上回荡。这一次，没有人反驳。

会议结束后，陆沉走到苏烬身边。"你做得不错。"

苏烬摇头。"还差得远。"

"但至少他们开始听了。"

苏烬点头。她看向远方——东方的地平线上，灰白色的天空下，一片死寂。管理者暂时没有动静，但苏烬知道那只是暴风雨前的宁静。

"我们需要更多的盟友。"她说，"铁锈镇单独扛不住管理者的攻击。"

"废土上还有其他聚落。"陆沉说。

"我知道。"苏烬的眼睛亮了起来，"我要召集他们。所有聚落，所有能战斗的人。我们要组建一个联盟——废土联盟。"

陆沉看着她。在这一刻，他看到了苏烬身上某种新的东西——不是变异带来的能力，而是一种从内心深处生长出来的力量。

"好。"他说，"我帮你。"
